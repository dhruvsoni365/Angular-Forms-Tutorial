### Prerequisites

### Installation

1. Navigate to the project directory:
```bash
cd evermore-app
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash

```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build for Production

```bash

```
